To Do Web App

Never forget what you have to in your day to day life with this To Do Web App. Add new your new tasks whenever you need to and delete them as you complete them. Along with the generic "Today" To Do List, this app allows you to create custom ToDo Lists by using Express Route Parameters. In order to make a custom list, all you have to do is type "/customListName" after the url. This customListName can be anything you like. e.g. Work or School.

HTML, CSS, Javascript, EJS, Command Line, Node.js, Express.js, MongoDB, Mongoose, Git, GitHub, Cyclic
